ProtocolAdapter
===============

Protocol Adapter is an Android library that manages at low-level the data collection from different sources. It makes use for Bluettoth HDP management of Signove's Antidote libraries available as well under L-GPL licence from ( http://oss.signove.com/index.php/Antidote:_IEEE_11073-20601_stack ) for what concerns the acquisition from HDP medical devices. 
This work was carried out with the support of the FI-STAR project (“Future Internet Social and Technological Alignment Research”), an integrated project funded by the European Commission through the 7th ICT - Framework Programme (318389).
